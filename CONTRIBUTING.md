# Contributors to Timeless Transport Models Technical Report

We are happy to have our team members contribute to the analysis of our developing company. There are many ways in which this project was created. Contributions include, data refinement, forecasting, data visualization, and analytical deductions.

## Contributing Code:

General workflow for contributing to this project

* Clone the repository
* Create a local branch to add contributions
* Commit your changes and push your created branch to the main repo
* Open a new pull request into our master branch
